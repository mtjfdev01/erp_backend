Bank Alfalah APG — ddr_server integration (mirrors PayFast pattern)
====================================================================

SETUP
-----
1. Set ALFALAH_ENV=sandbox | production
   - always required: APG_MERCHANT_ID, APG_STORE_ID, APG_MERCHANT_HASH, USERNAME, PASSWORD
   - sandbox only optional: APG_KEY1, APG_KEY2 (derived if omitted)
   - production: APG_KEY1 and APG_KEY2 required (16 chars each)
2. APG_BASE_URL auto-defaults from ALFALAH_ENV if omitted.
2. Set API_PUBLIC_BASE_URL to your public API host (HTTPS in production).
   Optional APG_HS_RETURN_URL — card step-1 return base (must match portal whitelist).
   Server appends ?donationId={id}. Examples:
     https://www.mtjfoundation.org/donate/alfalah-card  (preferred)
     https://www.mtjfoundation.org/thanks  (frontend redirects AuthToken → alfalah-card)
   BASE_Frontend_URL=https://www.mtjfoundation.org
3. In APG merchant portal: Return URL = {API}/donations/public/alfalah/return
   Listener URL = {API}/donations/public/alfalah/listener (whitelist with bank).
4. Run migration: src/database/migrations/CreateAlfalahPaymentSessions1760000000023.ts

CREATE DONATION (same as PayFast — POST /donations)
---------------------------------------------------
donation_method: "alfalah"
alfalah_transaction_type: "1" | "2" | "3"   (wallet | account | card)
alfalah_account_number: required for types 1 and 2 (wallet/account number)
donor_phone, donor_email: required (APG sends OTP/OTAC to these)

Response (wallet/account):
{
  "requiresOtp": true,
  "isOtp": true,           // true = 8-digit SMS OTP (wallet)
  "otpLength": 8,          // false = 4+4 SMS+email OTAC (account)
  "processOtpUrl": "/donations/public/alfalah/process-otp",
  "donationId": 123,
  "transactionReference": "123"
}

Response (card):
Preferred: cardStep 2 + SSO form (server REST handshake with channel 1001).
Fallback: cardStep 1 + POST to HS/HS/HS (if REST handshake fails — same as before).

Fallback (raw AuthToken on frontend):
GET /donations/public/alfalah/card-sso?donationId=&authToken=
→ { formAction, formFields } for frontend /donate/alfalah-card redirect.

SUBMIT OTP (after donor receives code from Bank Alfalah)
--------------------------------------------------------
POST /donations/public/alfalah/process-otp
Body (wallet):  { "donationId": 123, "smsOtp": "12345678" }
Body (account): { "donationId": 123, "smsOtac": "abcd", "emailOtac": "wxyz" }

On success: donation status = completed (also verified via APG IPN OrderStatus).

OTHER PUBLIC ROUTES
-------------------
POST /donations/public/alfalah/listener?url=<APG IPN GET URL>
GET  /donations/public/alfalah/return   (browser redirect → frontend thank-you)
GET  /donations/public/alfalah/status  (handshake health check)
GET  /donations/:id/provider-status    (polls APG when donation_method is alfalah)
